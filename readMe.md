## Installation
Follow these steps to get the project running locally:

1. **Clone the repository:**

   - ```bash
    git clone https://github.com/utibenoah/cineflixs.git

2. **Navigate to repository:**
    - ```bash
    cd cineflixs
3.  **Install dependencies** 
    - ```bash
    npm install
4.  **Run Project:**
    - ```bash
    npm start
